# Opciónes

## 1. Creación manual de la base de datos

- Crear base de datos con el nombre `packetworld`
- Importar base de datos `mexico.sql` a `packetworld`
- Ejecutar el script `DDLPacketWorld.sql`
- Ejecutar el script `DMLPacketWorld.sql`

## 2. Importación de la base de datos
- Crear base de datos con el nombre `packetworld`
- Importar base de datos `export.sql`
